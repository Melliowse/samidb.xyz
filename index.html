SamiDB
A very cozy discord bot.

A True Multi-Purpose Bot.

SamiDB is a stylish bot with many commands and features that are sure to make your server special.
With commands that to talk between different servers, emote commands, logging features, and more, you can make your server unique to you!

Aestheticly Pleasing

Most bots show a lot of information in a small space,
but Sami is not most bots, she was designed to be clean and compact without removing useful information.
When designing a command, we take into account all the weird inconsistencies on discord between platforms and make sure it looks right for everyone.
